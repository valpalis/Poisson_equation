# Poisson_equation

Проект представлен для решения задачи Пуассона в прямоугольнике с помощью параллельной верхней релаксации. 

# SOR 
Исходный алгоритм представлен в файле "sor.cpp". По сетке обновление значений искомой функции идет слева направо и снизу вверх.
В случае параллельной обработки необходимо избавиться от зависимости итераций цикла. Как вариант, можно разбить все узлы на "четные" и "нечетные". В таком случае параллельно обрабатывать сначала "четные" узлы, потом "нечетные". Параллельный алгоритм представлен в файле "parall_sor.cpp"

# Analys
Максимум итераций, который можем совершить, принял за K = 10000, в качестве критерия остановки взял L_2-норму меньше 0.000001. Сходимость можно проверить на трех последовательных приближениях. Точность проверял на точном решении.
