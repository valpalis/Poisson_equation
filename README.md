# Poisson_equation

Проект представлен для решения задачи Пуассона в прямоугольнике с помощью параллельной верхней релаксации. Более точно, уравнение Пуассона берется  на единичном квадрате с правой частью --- $-2\pi^2sin(\pi  x)sin(\pi  y)$. Граничные условия определены для искомой функции нулями на границе квадрата. Других условий нет.

# SOR 
Исходный алгоритм представлен в файле "sor.cpp". По сетке обновление значений искомой функции идет слева направо и снизу вверх.
В случае параллельной обработки необходимо избавиться от зависимости итераций цикла. Как вариант, можно разбить все узлы на "четные" и "нечетные". В таком случае параллельно обрабатывать сначала "четные" узлы, потом "нечетные". Параллельный алгоритм представлен в файле "parall_sor.cpp".

# Analysis
Максимум итераций, который можем совершить, принял за K = 10000, в качестве критерия остановки взял L_2-норму меньше 0.000001. Сходимость можно проверить на трех последовательных приближениях. Точность проверял на точном решении.

Погрешность - это разница между полученным приближенным решением и точным. N отвечает за количество узлов в сетке (Их в точности (N+1)^2, сетка равномерная). Как мы видим, число итераций и погрешность остаются неизменными при фиксированном N, меняется только время обработки цикла. Оно обратно пропорционально числу потоков.
